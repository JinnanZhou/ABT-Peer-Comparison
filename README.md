# ABT Peer Comparison

## Project Overview
This project analyses Abbott Laboratories (ABT) and compares its financial performance with the peer average using WRDS data.

## Research Questions
1. Is Abbott Laboratories (ABT) above or below the peer average in sales?
2. Is Abbott Laboratories (ABT) above or below the peer average in net income?
3. Is Abbott Laboratories (ABT) more or less profitable than the peer average?
4. Is Abbott Laboratories (ABT) larger or smaller than the peer average in terms of total assets?

## Files
- `track2_analysis.ipynb` – Jupyter Notebook containing the analysis
- `reflection_report.docx` – Reflection report
- `README.md` – Project description

## Data Source
The data used in this project was obtained from WRDS and analysed in Python.

## Summary
The analysis compares ABT with its peer group based on sales, net income, profitability, and firm size, and presents the findings with tables and visualisations.
